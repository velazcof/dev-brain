
**OAuth 2.0** is an **authorization framework** that enables secure, delegated access to resources without sharing user credentials.

---
### `Purpose`

To allow users to grant third-party applications limited access to their data (e.g., “Login with Google”) while keeping credentials private.

---
### `Structure`

- **Roles:** Resource Owner, Client, Resource Server, Authorization Server.
    
- **Grant Types:** Authorization Code, Client Credentials, Implicit, Resource Owner Password.
    
- **Tokens:** Access and Refresh Tokens define permissions and session validity.
