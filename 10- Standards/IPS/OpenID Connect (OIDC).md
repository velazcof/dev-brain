
**OpenID Connect (OIDC)** is an **authentication standard** built on top of **OAuth 2.0**, providing an identity layer for secure user login and federation.

---
### `Purpose`

To enable users to **authenticate with a trusted identity provider** (like Google or Microsoft) and access multiple applications with a single identity.

---
### `Structure`

- **ID Token:** JWT containing user identity information.
    
- **Authorization Endpoint:** Manages login and consent.
    
- **Token Endpoint:** Issues tokens (access, refresh, ID).
    
- **UserInfo Endpoint:** Returns user profile data.
