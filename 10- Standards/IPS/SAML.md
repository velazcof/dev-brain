
**SAML** **(Security Assertion Markup Language)** is an **XML-based open standard** for exchanging authentication and authorization data between an identity provider (IdP) and a service provider (SP).

---
### `Purpose`

To enable **Single Sign-On (SSO)** across enterprise systems by allowing users to authenticate once and gain access to multiple applications securely.

---
### `Structure`

- **Assertions:** XML statements carrying authentication and attribute information.
    
- **Bindings:** Define how SAML messages are transmitted (e.g., HTTP Redirect, POST).
    
- **Profiles:** Define use cases like web browser SSO.
    
- **Metadata:** Establish trust between IdPs and SPs.
