
Physical servers in the cloud that are **fully dedicated to a single customer**, ensuring **hardware-level isolation**.

---
### `Key Ideas`

- Used for **compliance, licensing, or performance isolation**.
    
- Eliminates the “noisy neighbor” issue of shared environments.
    
- More expensive but required for **BYOL (Bring Your Own License)** scenarios.
    
- Each provider has a variant (e.g., AWS Dedicated Host, Azure Dedicated Host, GCP Sole-Tenant Node).
