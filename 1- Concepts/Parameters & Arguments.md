
**Parameters and arguments** define how data is **passed into a function**.  
**Parameters** are the named placeholders declared by a function, while **arguments** are the actual values supplied when the function is called.

---
### `Key Ideas`

- **Declaration vs Invocation**
    
    - _Parameters_ belong to the function definition.
    - _Arguments_ belong to the function call.
    
- **Interface of a Function**  
    Parameters define what inputs a function expects and in what form.
    
- **Arity**  
    The number of parameters a function accepts.
    
- **Positioned or Named**  
    Some languages pass arguments by position, others by name, or both.
    
- **Default Values**  
    Parameters may define defaults, allowing arguments to be optional.
    
- **Variadic Parameters**  
    Some functions accept a variable number of arguments.
    
- **Type Association**  
    Parameters may specify expected data types, either explicitly or implicitly.
