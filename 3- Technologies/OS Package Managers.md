
**OS Package Managers** are tools that install, update, configure, and remove **system-level software** on an operating system. They manage applications, services, drivers, libraries, and system tools across the entire machine.

---
### `Key Ideas`

- **System-wide software management**, often requiring elevated privileges.
    
- **Integration with OS repositories**, delivering security updates and feature upgrades.
    
- **Dependency resolution** at the operating system level.
    
- **Uniform installation process** for applications, services, and utilities.
    
- Common implementations across OS families:
    
    - Linux: apt, dnf, pacman, zypper
    - macOS: Homebrew
    - Windows: Winget, Chocolatey

---
### `Use Cases`

- Installing and updating system applications and utilities.
- Applying OS security patches.
- Managing system services or drivers.
- Setting up development environments on Linux, macOS, or Windows.
- Ensuring consistent software across servers or workstations.
